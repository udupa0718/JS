JavaScript Encapsulation

It is process of binding the data with the functions acting on that data. It allows us to control the data and validate it.

Points to Remember..!
  • Use var keyword to make data members private.
  • Use setter method to set the data and getter method to get that data.

Encapsulation allows us to handle an object using the following properties.

read/Write - Here, we use setter methods to write the data and getter methods read the data.

Read Only -In this case, we use getter methods only.

Write Only -In this case, we use setter methods only.
